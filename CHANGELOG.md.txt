# Changelog
All notable changes to Ruby Market Scanner are documented here.


###[1.5.8] 2026-03-13

###Fixed
- Corrected PyInstaller output path mismatch that caused the installer to fail when copying application files.
- Updated Inno Setup [Files] section to reference the actual ONEDIR build directory (Ruby_GUI\*) instead of the non‑existent dist\Ruby_GUI\*.
- Resolved missing _internal runtime folder in installed builds, which previously caused python311.dll load failures.
- Eliminated EXE startup crashes and CMD flash behavior by ensuring the full ONEDIR runtime is bundled and installed correctly.

###Improved
- Build pipeline now produces a complete, stable ONEDIR distribution suitable for packaging and distribution.
- Installer now reliably deploys all required Python, Tcl/Tk, and dependency DLLs.
- Application launch is now clean and silent with no console flashes


###[1.5.7] 2026-03-12
### Fixed 
- Error scanning stock symbols with Alpha Vantage API
- Added skip note to unsupported symbols

###[1.5.6] – 2026‑03‑12
### Fixed
- Eliminated vertical spike artifacts in charts by implementing masked‑array handling for all indicators (RSI, MACD, ATR, Bollinger).
- Corrected Bollinger Band rendering to avoid drawing across invalid or nan regions.
- Fixed ATR band distortion caused by early‑series nan values.
- Resolved MACD/Signal/Histogram zero‑line artifacts by masking incomplete EMA windows.
- Cleaned Coinbase intraday data pipeline to remove malformed candles and stabilize indicator inputs.
### Improved
- Upgraded chart renderer to professional‑grade stability with full invalid‑value masking.
- Enhanced Bollinger Band plotting with validated index masking and consistent z‑ordering.
- Improved ATR shading and outline logic using masked arrays for clean transitions.
- Refined volume subplot scaling behavior for clearer visual interpretation.
- Strengthened indicator pipeline alignment with cleaned candle data.
### Added
- Robust wick‑based outlier detection for crypto candles to prevent ATR/BB contamination.
- Composite signal engine now produces coherent, stable bias and confidence scoring.
- VWAP integration now fully stable across all cadences.
